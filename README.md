# Startup-Trends-SQL-Analysis
This SQL project explores trends in the startup ecosystem using a dataset of startup companies stored in a relational database. The analysis focuses on key business metrics such as company valuation, funding raised, founding year, market category, and employee size.
Startup Trends SQL Analysis
Overview

This project explores trends in the startup ecosystem using SQL and MySQL. The dataset contains information on startup companies, including their location, category, employee count, funding raised, valuation, founding year, stage, CEO, and company description.

The goal of the project was to analyze the data using aggregate functions and grouping techniques to answer business questions about startup growth, market competitiveness, funding, and company size.

Objectives

In this project, I:

Created a MySQL database and table for startup data

Inserted startup records into the table

Wrote SQL queries to analyze funding, valuation, age, category, and location trends

Used aggregate functions to generate business insights from the dataset
